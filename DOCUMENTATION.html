Introduction

Thank you for purchasing ARCHDesign Architects! This documentation will guide you through installing, updating, and customizing your theme for your architectural portfolio or business website. All images are included locally, and the template is fully self-contained for easy deployment.

Installation

  1. Unzip the downloaded package from CodeCanyon.
  2. Upload the entire theme folder to your web server or hosting account using FTP, SFTP, or your hosting file manager.
  3. Set the root directory (where index.html is located) as the public web root, or place the files in a subfolder as needed.
  4. Access your site in a browser by navigating to your domain or the subfolder where you uploaded the files.
Note: No installation of CMS or database is required. This is a static HTML template.

Updating the Theme

  1. Backup your current theme folder and any files you have customized.
  2. Unzip the new version of the theme.
  3. Replace the old files with the new ones.
    Tip: Only overwrite files you have not customized, or manually merge your changes if needed.
  4. Clear your browser cache and refresh your site to see the updates.

Customization

Editing Text Content

  1. Open the relevant .html file (e.g., index.html, careers.html, or any file in case-studies/).
  2. Locate the text you wish to change. You can use any text editor (VS Code, Sublime Text, Notepad++, etc.).
  3. Edit the text between the HTML tags. For example: 
    <h1>ARCHDesign Architects</h1>
<p>Crafting spaces that transcend the ordinary</p>
  4. Save the file and refresh your browser to see the changes.

Changing Images

  1. All images are stored in the images/ folder and its subfolders.
  2. To replace an image, overwrite the existing file with your new image (keep the same filename for instant update), or:
    • Upload your new image to the appropriate subfolder (e.g., images/portfolio/).
    • Update the src or background-image path in the HTML file. Example: 
      <img src="images/portfolio/my-new-image.jpg" alt="Description">
  3. For hero/background images, update the style attribute: 
    <div class="hero-image" style="background-image: url('images/hero/my-hero.jpg')"></div>

Changing Colors & Styles

  1. Open styles.css in a text editor.
  2. Find the CSS variable or selector you want to change (e.g., body, .header, .footer).
  3. Edit color codes, font sizes, or other style properties as desired.
  4. Save the file and refresh your browser to see the new styles.

Navigation & Links

  1. Navigation menus are defined in the <nav> sections of index.html and other HTML files.
  2. To add, remove, or rename menu items, edit the <ul> and <li> elements inside the navigation.
  3. Update the href attributes to point to the correct pages or sections.

FAQ & Troubleshooting

  • Images not showing?
    Ensure the image path is correct and the file exists in the images/ folder. Paths are relative to the HTML file's location.
  • Fonts not loading?
    The template uses Google Fonts. Ensure you have an internet connection, or self-host fonts if needed.
  • Broken layout?
    Check that styles.css is present and linked correctly in your HTML files.
  • JavaScript not working?
    Make sure script.js is present and linked at the end of the <body> tag.
